# Project

This project was created using the **Next.js Docker Development Template**.

It provides a complete local development environment running inside Docker.

---

# Development Environment

The stack may include:

- Next.js
- Docker
- PostgreSQL
- Redis
- pgAdmin
- Redis Commander

All services run inside Docker containers so **no database or cache services need to be installed on your system**.

---

# Start Development

Build and start the development environment:

```bash
./dev.sh rebuild
```

---

# Check Running Services

Ports may change automatically if default ports are already in use.

To see the correct service URLs run:

```bash
./dev.sh ports
```

---

# Development Commands

Start environment

```bash
./dev.sh start
```

Stop environment

```bash
./dev.sh stop
```

Rebuild containers

```bash
./dev.sh rebuild
```

View logs

```bash
./dev.sh logs
```

Reset environment

```bash
./dev.sh reset
```

---

# Environment Variables

Environment configuration is stored in:

```
.env
```

This file is automatically generated during project setup.

# Notes

- All services run inside Docker containers.
- The development environment is isolated from your system.
- Ports are automatically assigned if defaults are already in use.
- Optional services such as PostgreSQL, Redis, pgAdmin and Redis Commander may not be present if they were disabled during setup.
